# Web Directory Enumeration using Gobuster

## Objective
Discover hidden directories and files on a web server.

## Tools Used
- Gobuster
- Kali Linux

## Command Used
gobuster dir -u http://target-site -w wordlist.txt

## Description
Performed directory brute forcing using Gobuster to identify hidden directories and files on a web server.

This technique helps security professionals identify exposed resources that may lead to security vulnerabilities.

## Outcome
Successfully discovered hidden directories and potential attack surfaces within the web application.

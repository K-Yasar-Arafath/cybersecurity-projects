# Wireshark Traffic Analysis

## Objective
Capture and analyse network traffic to understand communication between devices.

## Tools Used
- Wireshark
- Kali Linux / macOS

## Description
Captured live network traffic using Wireshark and analysed packets to identify protocols such as DNS, TCP, and HTTP.

This helps security analysts understand network behaviour and detect suspicious activity.

## Key Learning
Learned how to inspect packet details, including source IP, destination IP, protocol type, and packet structure.

## Outcome
Successfully captured and analysed network traffic to understand communication patterns within the network.

# Phishing Email Investigation

## Objective
Analyze a suspicious email to identify phishing indicators.

## Tools Used
- Email header analysis
- VirusTotal

## Description
Investigated a phishing email example to identify malicious indicators such as suspicious sender address, urgent language, and potentially harmful links.

## Indicators Found
- Suspicious sender email address
- Urgent request for account verification
- Suspicious domain link

## Outcome
Successfully identified phishing characteristics and verified domain reputation using VirusTotal.

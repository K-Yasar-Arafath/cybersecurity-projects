# Network Scanning using Nmap

## Objective
Identify open ports and services running on a target machine.

## Tools Used
- Nmap
- Kali Linux

## Command Used
nmap -sV target-ip

## Description
Performed network scanning to discover open ports and running services using Nmap.

This helps identify potential attack surfaces in a network.

## Outcome
Successfully detected open ports and service versions running on the target system.

# Threat Intelligence Lookup

## Objective
Investigate a suspicious IP address using threat intelligence platforms.

## Tools Used
- VirusTotal
- AbuseIPDB

## Indicator Investigated
IP Address: 185.199.110.153

## Investigation

The IP address was analyzed using VirusTotal to determine whether it was associated with malicious activity.

VirusTotal Results:
- 1 out of 94 security vendors flagged the IP as malicious.
- Most vendors classified the IP as clean.

Additional Information:
- ASN: AS54113
- Hosting Provider: Fastly Inc.
- Country: United States

Crowdsourced intelligence suggested possible activity related to Metasploit testing infrastructure.

## Conclusion

The IP address showed low malicious detection and may be associated with shared hosting infrastructure. 

Based on the available intelligence, the risk level is considered low. Continuous monitoring is recommended rather than immediate blocking.

## Outcome

Threat intelligence platforms help security analysts quickly determine the reputation of suspicious IP addresses and assist in security investigations.
